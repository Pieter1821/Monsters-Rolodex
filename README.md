

# Monsters-Rolodex

This project is a simple web application that displays a list of monsters and their details.

## Prerequisites

- Node.js (version 18+)
- Yarn (version 1.22+)
- npm (version 7.24+)

## Installation

1. Clone the repository.
2. Install dependencies by running `yarn install` or `npm install`.
3. Start the development server by running `yarn dev` or `npm run dev`.
4. Build the project by running `yarn build` or `npm run build`.

## Usage

- `yarn dev` or `npm run dev`: Starts the development server with hot reloading.
- `yarn build` or `npm run build`: Builds the project for production.
- `yarn serve` or `npm run serve`: Serves the production build.
